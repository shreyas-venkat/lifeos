Write a follow-up email. Context: $ARGUMENTS

If a thread ID or subject is provided, fetch the original email via Gmail MCP to understand context.

Write a follow-up that:
- References the original email naturally ("Following up on my email from <timeframe>...")
- Restates the key ask in one sentence — don't make them go searching
- Is not passive-aggressive or apologetic for following up
- Provides a clear, easy path to respond (yes/no question, or explicit deadline)
- Is shorter than the original email

**Tone guidelines by wait time:**
- < 3 days: friendly, brief
- 3–7 days: polite but more direct about needing a response
- > 7 days: direct, reference the deadline if there is one, offer an alternative

If $ARGUMENTS specifies a deadline or meeting, anchor the urgency to that.

Present for review before sending.
